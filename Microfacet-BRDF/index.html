Cook-Torrance Microfacet BRDF

$$f(l,v) = \frac{D(h)F(l,h)G(l,v,h)}{4(n\cdot l)(n\cdot v)}$$

  • D: 法线分布项
  • G: 几何遮蔽项
  • F: 菲涅尔项
  • l: 入射光方向
  • n：法线方向
  • h: 半向量,入射方向与观察方向的合向量

法线分布项

GGX

$$D(h) = \frac{\alpha^2}{\pi((n\cdot h)^2(\alpha^2 - 1) + 1)^2}$$

  • $\alpha$: $roughness^2$
  • n: 法线
  • h: 半向量,入射方向与观察方向的合向量方向

几何遮蔽项

Smith

$$G(l,v,h) = G_1(l)G_1(v)$$
$$G_1(v) = \frac{n\cdot v}{(n\cdot v)(1 - k) + k}$$
$$k = \frac{(Roughness + 1)^2}{8}$$

  • v: 观察方向
  • n: 法线方向
  • l: 入射方向

菲涅尔项

Shclick

$$F_{schlick}(F_0,l,h) = F_0 + (1 - F_0)(1-(l\cdot h))^5$$

  • $F_0$: 0度(法线与视线平行)时的菲涅尔值,也表示高光颜色
