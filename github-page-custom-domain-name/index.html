github page custom domain name

2017-11-21

域名注册

登录GoDaddy,选择域名进行购买

DNS解析

注册腾讯云,腾讯云可以免费解析域名

添加完之后将会看到解析状态为域名DNS未更改,需要登录GoDaddy修改DNS服务器

域名服务器填写f1g1ns1.dnspod.net,f1g1ns2.dnspod.net

回到腾讯云,解析域名,添加以下两条记录

主机记录 记录类型 线路类型 记录值 TTL
@ CNAME 默认 <your github name>.github.io 600
www CNAME 默认 <your github name>.github.io 600

Github Page配置

上传一个CNAME文件在Github Page根目录下,其内容为你的域名

hexo每次部署都是将本地文件覆盖远程,因此,需要在本地的source文件夹中存放CNAME文件

CNAME没有后缀名
